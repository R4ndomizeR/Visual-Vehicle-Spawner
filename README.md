# Visual-Vehicle-Spawner 1.0 for alt:V
Visual Vehicle Spawner 1.0 for alt:V 

## How to install?

1. Unpack .rar file
2. rename 'Visual-Vehicle-Spawner-master' to 'vvs'
3. Put 'vvs' folder to your 'resources' folder
4. in server.cfg add 'vvs' in "resources"
